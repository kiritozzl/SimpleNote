# SimpleNote
简单的笔记小应用，可以增加，修改，删除笔记，笔记会记录添加，修改笔记的时间；
添加了笔记闹钟提醒功能，设定提醒的时间，笔记内容显示在闹钟上，到设定时间时会唤醒屏幕，
震动三下，播放音乐。
